1 #Description
2 Easy SHM library for C++, still a work in progress but it's
3 a easier way to code up an anonymous SHM mapping, protect
4 the last page and generate a key to access it....I've used
5 similar code quite a bit and was tired of re-coding it. If
6 I find time I'll make it a bit more advanced, like adding
7 a monitor process to cleanup if for some reason the code
8 crashes (you know, things happen right?). I've tested the
9 code on OS X and Linux, but as usual your mileage may vary
10 greatly.

21 #Compilation Notes
22 To build this library on OS X you'll need to run:
23 glibtoolize
24 aclocal
25 autoconf
26 automake --add-missing
27 make
28 make check
29 sudo make intall
30 
31 The only changes for linux will be libtoolize for
32 call to libtool instead of glibtoolize...on OS X
33 the binary might be named differently depending
34 on how you've installed it so your mileage may
35 vary. If you're advanced enough to install libtool
36 on OS X you should be able to figure this out.

39 #Usage
40 To use this library, simply:
41 \#include <shm>
42 
43 When compiling your code that uses it, link with -lshm. On
44 Linux you'll have to compile with the -lrt, -lpthread and
45 -lnuma in order to fully utilize all the features. In
46 the future I might add a flag to compile without the NUMA
47 features so that you can use the library without having
48 to install libnuma.
